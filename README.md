# Finance Tracker 💰

A simple Python project to track and manage personal finances.  
It records income and expenses, categorizes transactions, and provides clear summaries to help you understand your financial health.

## Features
- Add and manage income and expenses
- Categorize transactions (e.g., Food, Rent, Entertainment)
- View summary statistics:
  - Total Income
  - Total Expenses
  - Net Balance
- Export or display reports in a clean format
- Menu‑driven interface for easy use

## Requirements
- Python 3.x
- pandas
