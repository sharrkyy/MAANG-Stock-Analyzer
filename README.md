MAANG Stock Analyzer

Overview

This Python program provides an interactive command-line tool to analyze and visualize stock data for the MAANG companies (Meta, Apple, Amazon, Netflix, Google). It reads stock data from a CSV file, summarizes key statistics, and generates candlestick charts using mplfinance.

Features

Interactive menu to select a company (Meta, Apple, Amazon, Netflix, Google).

Filters stock data by ticker symbol.

Displays summary statistics:

Highest High

Lowest Low

Plots candlestick charts with optional volume overlay.

Custom color palette and style for improved visualization.

Looping menu with replay/exit options.

Requirements

Python 3.x

Libraries:

pandas

mplfinance

Install dependencies with:

pip install pandas mplfinance
